🚀 getButton 基本使用範例

這是一個輕量級的 Line QR Code 按鈕組件，可以輕鬆嵌入任何網站。

1. 基本使用

只需要引入腳本和添加容器元素：

<!-- 引入腳本 -->
<script src="../dist/getButton.js"></script>

<!-- 添加按鈕容器 -->
<div id="line-button"></div>

結果：

2. 自訂設定

可以透過 window.getButtonConfig 自訂設定：

<script>
window.getButtonConfig = {
  qrCodeImage: './custom-qr.png',
  lineUrl: 'https://lin.ee/your-line-id',
  buttonText: 'Line 聯絡我們',
  theme: 'white'
};
</script>

結果（白色主題）：

3. 功能特色

  • ✅ 輕量級，無外部依賴
  • ✅ 響應式設計，支援手機和桌面
  • ✅ Line 官方品牌風格
  • ✅ 支援 ESC 鍵關閉彈窗
  • ✅ 點擊遮罩關閉彈窗
  • ✅ 平滑動畫效果
