💡 基本使用

只需要引入腳本，按鈕會自動出現在右下角：

<!-- 引入腳本 --> <script src="getButton.js"></script> <!-- 就這樣！按鈕會自動出現在右下角 -->

🎛️ 自訂設定

可以透過設定來自訂按鈕的行為：

<script> // 自訂設定 window.getButtonConfig = { qrCodeImage: './my-qr-code.png', lineUrl: 'https://lin.ee/your-line-id', buttonText: 'Line 聯絡我們', theme: 'white', position: 'fixed', // 固定位置模式 bottom: '20px', // 距離底部 20px right: '20px' // 距離右邊 20px }; </script> <script src="getButton.js"></script>

設定選項：

選項 類型 預設值 說明
position string 'fixed' 位置模式：'fixed' 或 'inline'
bottom string '20px' 距離底部的距離（fixed 模式）
right string '20px' 距離右邊的距離（fixed 模式）
theme string 'green' 主題：'green' 或 'white'
buttonText string 'Line' 按鈕顯示文字
qrCodeImage string './src/assets/line-qr.svg' QR Code 圖片路徑
lineUrl string 'https://lin.ee/mk5SfVU' Line 連結網址

📋 內聯模式演示

如果你想要按鈕顯示在特定位置而不是浮動，可以使用內聯模式：

<script> window.getButtonConfig = { position: 'inline' // 內聯模式 }; </script> <script src="getButton.js"></script> <div id="line-button"></div>
