🔥 多按鈕模式

同時使用 Line 和電話按鈕

📱 功能展示

右下角會同時顯示兩個按鈕：

💚
Line 按鈕

點擊顯示 QR Code

📞
電話按鈕

點擊直接撥號

🛠️ 多按鈕設定方式

使用 window.getButtonConfigs 陣列來設定多個按鈕：

window.getButtonConfigs = [ { type: 'line', lineUrl: 'https://lin.ee/mk5SfVU', qrCodeImage: '../src/assets/line-qr.png', buttonColor: '#00c300', headerColor: '#00c300', position: 'fixed' }, { type: 'phone', phoneNumber: '+886912345678', buttonColor: '#e74c3c', position: 'fixed' } ];

⚙️ 進階設定

你可以為每個按鈕設定不同的位置：

window.getButtonConfigs = [ { type: 'line', position: 'fixed', bottom: '100px', // Line 按鈕在最底部 right: '20px' }, { type: 'phone', position: 'fixed', bottom: '20px', // 電話按鈕在上方 80px right: '20px' } ];

📋 內聯模式

也可以將按鈕放在頁面中的特定位置：

// 內聯按鈕設定 window.getButtonConfigs.push( { type: 'line', position: 'inline', targetId: 'line-inline' }, { type: 'phone', position: 'inline', targetId: 'phone-inline' } );
