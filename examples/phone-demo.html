📞 電話按鈕

getButton 電話功能測試

📱 測試說明

• 右下角會出現紅色圓形電話按鈕

• 點擊按鈕會直接撥打電話號碼

• 在手機上會開啟撥號程式

• 在桌面上會嘗試開啟預設的通話應用程式

電話按鈕設定

window.getButtonConfig = { type: 'phone', phoneNumber: '+886912345678', buttonColor: '#e74c3c', position: 'fixed' };

設定類型為 'phone' 即可啟用電話功能

自訂設定選項

{ type: 'phone', // 按鈕類型 phoneNumber: '電話號碼', // 要撥打的號碼 buttonColor: '按鈕顏色', // 按鈕背景色 buttonText: '按鈕文字', // 可選，預設為 'Phone' position: 'fixed' // 位置模式 }

支援國際號碼格式，建議加上國碼
